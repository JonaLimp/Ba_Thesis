## Get started

To get started do:
```bash
make venv
source venv/bin/activate
make install
```

To be sure that the code is working do:
```bash
make test
```

When editing the code do this command to format the code:
```bash
make black
```

When something weird happens and for cleanup:
```bash
make clean
```
